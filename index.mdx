Connector is a Clojure HTTP service that provides a unified API for loading CSV data from multiple cloud storage providers and local filesystems. Backed by Apache Spark, it reads your data and returns a preview of the first 10 rows in a single POST request.

## How it works

<Steps>
  <Step title="Build and start the service">
    Build the uberjar and start the Connector HTTP server on port 3000.

    ```bash
    lein uberjar
    lein run
    ```
  </Step>
  <Step title="Send a POST request">
    Call `POST /data-load` with a JSON body specifying the source type, file location, and any credentials.

    ```bash
    curl -X POST http://localhost:3000/data-load \
        -H "Content-Type: application/json" \
        -d '{
            "type": "local",
            "format": "csv",
            "link-type": "public",
            "source": { "path": "/data/input.csv" },
            "options": { "header": true, "delimiter": "," }
        }'
    ```
  </Step>
  <Step title="Receive a preview response">
    Connector returns a JSON response with the first 10 rows of your CSV data.

    ```json
    {
        "status": "success",
        "source": "local",
        "duration-ms": 123,
        "data": [
            { "columnA": "value1", "columnB": "value2" }
        ]
    }
    ```
  </Step>
</Steps>

## Supported sources

| Source | Public | Private |
|--------|--------|---------|
| Local filesystem | Yes | — |
| Google Drive | Yes | Yes (OAuth2) |
| Dropbox | Yes | Yes (token) |
| Amazon S3 | Yes | Yes (IAM keys) |
| Google Cloud Storage | Yes | Yes (service account) |
